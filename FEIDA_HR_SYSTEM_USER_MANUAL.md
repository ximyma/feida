## 一、系统概述

### 1.1 系统简介

**飞达智能HR系统**是一款综合性的人力资源管理系统，旨在帮助企业高效管理员工信息、考勤、薪资、绩效等核心业务。

### 1.2 系统特点

- 🎯 **功能全面**：涵盖人事管理、考勤管理、薪资核算、绩效考核等核心模块
- 📱 **界面友好**：采用现代化设计，操作简单直观
- 🔒 **安全可靠**：完善的权限管理和数据加密机制
- 📊 **数据可视化**：丰富的报表和数据分析功能

### 1.3 运行环境

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| 浏览器 | Chrome 90+、Edge 90+、Firefox 88+ |
| 网络 | 无需联网（单机部署） |

## 二、快速入门

### 2.1 启动系统

**方式一：脚本启动（生产模式，推荐）**
1. 双击 `启动系统.bat`（Windows）。它会在缺少依赖时自动 `npm install --legacy-peer-deps`、缺少构建时自动 `npm run build`，随后启动服务。
2. 启动后浏览器访问 **`http://localhost:3000`**（前端 SPA 与 API 同端口）。

**方式二：命令行**
```bash
npm install --legacy-peer-deps     # 安装依赖
npm run build                      # 构建
npm start                          # 启动（默认 3000 端口）
# 或开发模式：npm run dev  → 前端 http://localhost:8080，API http://localhost:3000/api
```
> 自定义端口：`SERVER_PORT=3400 npm start`。

### 2.2 登录系统

1. 在登录页面输入账号信息：
   - **用户名**：`admin`
   - **密码**：`admin123`
2. 点击「登录」按钮进入系统

> 其他预置账号见文末「附录：系统预置账号」。

### 2.3 系统主界面

登录后进入系统主界面，包含以下区域：

| 区域 | 说明 |
|------|------|
| **顶部导航** | 公司名称、用户信息、退出按钮 |
| **左侧菜单** | 功能模块导航菜单 |
| **主内容区** | 当前模块的操作界面 |
| **快捷入口** | 常用功能的快速访问入口 |

## 三、功能模块详解

### 3.1 首页仪表盘

首页仪表盘展示系统概览信息：

**主要功能：**
- 📊 **KPI统计卡片**：员工总数、在职人数、本月入职/离职、待审批、工资总额
- ⚡ **快捷入口**：快速跳转到常用功能模块
- 🏢 **部门人员分布**：饼图展示各部门人员占比
- 📈 **入职趋势**：近6个月入职趋势图表
- 🔔 **待办提醒**：待审批、合同到期、生日提醒
- 📢 **公告通知**：最新的公司公告

### 3.2 组织架构

管理企业组织架构信息：

**主要功能：**
- **组织架构图**：树形展示公司层级结构
- **部门管理**：新增、编辑、删除部门，支持部门划转和合并
- **岗位管理**：管理各部门的岗位设置
- **职级体系**：定义职级等级和薪资范围
- **编制管理**：查看和调整部门及岗位的编制情况

**操作说明：**
1. 点击左侧树形菜单查看部门详情
2. 点击「新增部门」创建新部门
3. 使用「部门划转」调整部门层级关系
4. 使用「部门合并」合并多个部门

### 3.3 人事管理

管理员工基本信息：

**主要功能：**
- **员工档案**：查看和编辑员工详细信息
- **员工花名册**：员工列表展示和筛选
- **合同管理**：劳动合同的签订、到期提醒、续签管理
- **员工异动**：部门调动、岗位调整、薪资变动
- **证件管理**：员工证件的上传和管理

**操作说明：**
1. 点击「员工档案」查看员工详情
2. 使用搜索框快速定位员工
3. 点击「新增员工」添加新员工信息
4. 在「合同管理」中设置合同到期提醒

### 3.4 考勤管理

管理员工考勤相关事务：

**主要功能：**
- **考勤打卡**：员工上下班打卡
- **请假管理**：请假申请、审批流程
- **加班管理**：加班申请和审批
- **班次管理**：定义不同的班次类型
- **考勤统计**：日报、周报、月报统计分析

**操作说明：**
1. 员工通过「考勤打卡」模块进行打卡
2. 申请请假需填写请假类型和时间
3. 加班需提前申请并经审批通过
4. 管理员可在「考勤统计」中查看考勤报表

### 3.5 薪酬管理

处理薪资核算和发放：

**主要功能：**
- **薪资查询**：查看员工薪资明细
- **薪资配置**：设置薪资项目和计算公式
- **社保管理**：社保缴纳和公积金管理
- **薪资调整**：员工薪资调整申请和审批

**操作说明：**
1. 在「薪资查询」中查看个人薪资明细
2. 管理员在「薪资配置」中定义薪资项目
3. 设置社保基数和缴纳比例
4. 通过「薪资调整」流程调整员工薪资

### 3.6 绩效管理

管理员工绩效考核：

**主要功能：**
- **考核周期**：定义考核周期和考核方案
- **KPI管理**：设置关键绩效指标
- **绩效评估**：进行员工绩效评分
- **绩效报表**：查看绩效分析报告

**操作说明：**
1. 创建考核周期并设置考核时间
2. 定义KPI指标和权重
3. 在考核周期内进行绩效评估
4. 查看绩效分布和分析报告

### 3.7 招聘管理

管理招聘流程：

**主要功能：**
- **职位发布**：发布招聘职位
- **简历管理**：收集和筛选简历
- **面试安排**：安排面试和记录评价
- **录用管理**：发送录用通知

**操作说明：**
1. 发布招聘职位并设置要求
2. 查看候选人简历并进行筛选
3. 安排面试时间并记录评价
4. 向合适候选人发送录用通知

### 3.8 培训管理

管理员工培训和学习：

**主要功能：**
- **课程中心**：管理培训课程
- **学习管理**：员工学习进度追踪
- **考试管理**：在线考试和测评
- **培训报表**：培训效果统计

**操作说明：**
1. 管理员创建培训课程
2. 员工在「我的学习」中选择课程
3. 完成课程学习并参加考试
4. 查看培训完成情况和成绩

### 3.9 审批中心

处理各类审批流程：

**主要功能：**
- **待审批**：需要处理的审批申请
- **已审批**：已处理的审批记录
- **申请记录**：个人提交的申请
- **工作流设计**：自定义审批流程

**操作说明：**
1. 在「待审批」中查看需要处理的申请
2. 点击申请查看详情并进行审批
3. 支持通过、拒绝、转签等操作
4. 管理员可设计自定义审批流程

### 3.10 综合事务

处理日常办公事务：

**主要功能：**
- **通知公告**：发布和查看公司公告
- **文档管理**：上传和共享文档
- **问卷调查**：创建和参与问卷
- **用车管理**：车辆申请和使用记录

**操作说明：**
1. 管理员在「通知公告」中发布公告
2. 员工可查看公告并标记已读
3. 在「文档管理」中上传和下载文件
4. 参与公司发起的问卷调查

### 3.11 系统管理

系统设置和维护：

**主要功能：**
- **用户管理**：管理系统用户、重置密码、分配角色
- **角色管理**：定义角色、配置**细粒度权限点**（如 `cms:article:delete`、`shop:goods:manage`）与**站点作用域**（`main` 主站 / `shop` 商城 / `portal` 门户资讯，`["*"]` 为全部）
- **系统配置**：系统参数设置（公司名称、发薪日、主题等）
- **数据管理**：数据库备份与恢复
- **AI 配置**：配置 AI 服务提供商与 API Key

**操作说明：**
1. 管理员可新增和编辑用户，分配一个或多个角色
2. 在「角色管理」中勾选权限点，并设置该角色可管理的站点范围
3. 配置系统参数后即时生效
4. 定期备份数据库确保数据安全（亦可直接复制 `data/ehr.db` 文件）

## 四、常见操作指南

### 4.1 如何修改个人信息？

1. 点击右上角用户头像
2. 选择「个人设置」
3. 修改个人信息后点击「保存」

### 4.2 如何申请请假？

1. 进入「考勤管理」→「请假管理」
2. 点击「新增请假申请」
3. 填写请假类型、时间、原因
4. 提交申请等待审批

### 4.3 如何查看薪资明细？

1. 进入「薪酬管理」→「薪资查询」
2. 选择查询月份
3. 点击查看详情

### 4.4 如何发布公告？

1. 进入「综合事务」→「通知公告」
2. 点击「新增公告」
3. 填写标题、内容、类型
4. 点击「发布」（或保存为草稿）

### 4.5 如何审批申请？

1. 进入「审批中心」→「待审批」
2. 选择需要处理的申请
3. 查看申请详情
4. 点击「通过」或「拒绝」，可添加审批意见

## 五、常见问题

### Q1：忘记密码怎么办？

**A**：请联系系统管理员重置密码。

### Q2：无法登录系统？

**A**：请检查：
- 服务器是否正在运行（查看 `start.bat` 窗口）
- 浏览器地址是否正确（应为 `http://localhost:8080`）
- 账号密码是否正确

### Q3：如何备份数据？

**A**：
1. 进入「系统管理」→「数据管理」
2. 点击「数据库备份」
3. 选择备份路径并确认

### Q4：公告不显示在首页？

**A**：公告需要先「发布」才能显示在首页仪表盘，草稿状态的公告不会显示。

### Q5：如何添加新员工？

**A**：
1. 进入「人事管理」→「员工档案」
2. 点击「新增员工」
3. 填写员工基本信息
4. 点击「保存」

## 七、AI智能功能

### 7.1 AI智能助手

**入口**: 侧边栏「🤖 AI智能 → AI助手」或页面右下角浮动按钮

**主要功能**：
- 💬 **智能对话**：自由提问HR相关问题，AI即时回答
- 📚 **知识库增强**：开启后AI会自动搜索知识库作为回答依据
- 🎭 **模型切换**：支持 DeepSeek V3 / R1 / GPT-4o Mini 三种模型
- 🔧 **工具模式**：支持搜索、分析、翻译、文档提取四种专业模式
- 📁 **对话管理**：左侧面板查看/切换/删除历史对话
- ⚡ **流式输出**：回答实时显示，打字机效果

**快捷操作**：
- 数据分析：AI分析考勤、薪酬等HR数据
- 招聘建议：获取招聘效率提升建议
- 制度查询：查询HR政策和规章制度
- 翻译辅助：多语言翻译（中/英/日/越/柬等）

### 7.2 知识库管理

**入口**: 「AI智能 → 知识库」

**主要功能**：
- 📚 **知识条目管理**: 添加/编辑/删除HR知识（制度、流程、规范）
- 🔍 **混合搜索**: 语义搜索 + 关键词搜索双重检索
- 🎯 **命中测试**: 输入查询文本，查看检索效果和相关度评分
- 📂 **分类管理**: 8大分类（人事制度/薪酬福利/考勤管理/招聘入职/培训发展/绩效考核/员工关怀/通用知识）

### 7.3 智能BI分析

**入口**: 「AI智能 → BI分析」

**主要功能**：
- 📊 **核心指标**：员工总数、出勤率、平均薪资、绩效达标率
- 📈 **可视化图表**：考勤趋势柱状图、部门结构饼图、人员流动折线图
- 🤖 **AI洞察**：点击「AI分析」按钮获取智能数据解读

### 7.4 智能预警

**入口**: 「AI智能 → 智能预警」

**5类预警规则**：
| 预警类型 | 触发条件 |
|----------|----------|
| 连续迟到预警 | 员工连续迟到≥3次 |
| 加班超时预警 | 月加班时长>36小时 |
| 合同到期提醒 | 劳动合同<30天到期 |
| 绩效下滑预警 | 连续2月绩效下降 |
| 培训完成率预警 | 完成率低于70% |

### 7.5 AI系统配置

**入口**: 「系统管理 → AI配置」

**配置项**：
- 🔌 **服务提供商**: DeepSeek / OpenAI / Ollama（三选一）
- 🔑 **API Key**: 输入对应平台的API密钥
- 🧪 **连接测试**: 验证AI服务是否正常
- ⚙️ **高级参数**: Temperature（创意度）、Max Tokens（回复长度）

## 八、网站与商城

### 8.1 网站管理

**入口**: 「网站管理 → 网站管理」

**主要功能**：
- 📰 **栏目管理**: 创建树形栏目结构
- 📝 **文章管理**: 发布/编辑/删除文章
- 🎨 **Banner管理**: 轮播图管理
- 💬 **评论管理**: 文章评论审核
- ⚙️ **网站配置**: 网站名称/Logo/版权设置

### 8.2 商城管理

**入口**: 「商城管理 → 商城管理」

**主要功能**：
- 🛍️ **商品管理**: 添加/编辑/删除商品，设置价格库存
- 📦 **订单管理**: 查看/处理/发货/完成订单
- ⭐ **评价管理**: 商品评价审核管理

### 8.3 商城前台

**访问地址**: `/shop`

**功能**：商品浏览、分类筛选、购物车、下单支付、用户中心

### 8.4 网站前台

**访问地址**: `/site`

**功能**：文章阅读、新闻浏览、Banner轮播

### 8.5 权限控制与多语言

- **按钮级权限**：网站/商城后台的「添加 / 编辑 / 删除 / 批量操作 / 评论审核」等按钮按当前账号的**细粒度权限点**自动显隐。例如仅拥有 `cms:article:delete` 的角色可见「删除文章」按钮；普通员工账号进入后台时大部分操作按钮不可见。
- **多语言切换**：点击界面右上角的语言切换（中文 / English），全站界面文案实时切换。后台管理、商城、门户均支持中英双语。
