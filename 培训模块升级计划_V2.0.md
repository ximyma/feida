**版本：** V2.0  
**日期：** 2026-04-29  
**状态：** 规划中

---

## 一、升级目标

将现有培训模块升级为完整的**在线培训学习管理系统（LMS）**，支持：

1. **文字课程** - 图文课件、文档教材、在线阅读
2. **视频课程** - 点播学习、断点续看、进度追踪
3. **视频直播** - 实时直播、弹幕互动、录制回放

### 3.2 学习中心

#### 3.2.1 我的学习
```
我的学习
├── 待学习（必修课程）
│   ├── 课程卡片（封面、进度、截止日期）
│   └── 批量开始学习

│   ├── 继续学习（断点续学）
```

#### 3.2.2 学习记录功能
- **学习轨迹**：记录每个章节的开始/结束时间、观看时长

- **学习时长**：统计每日/每周/每月学习时长
- **笔记功能**：在课程中做笔记、划重点、收藏内容
- **问答讨论**：在学习过程中提问、查看其他学员的问题

## 四、数据库设计

### 4.1 核心表结构

#### 4.1.1 课程主表
```sql
-- 课程主表
CREATE TABLE training_courses (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title VARCHAR(200) NOT NULL,           -- 课程标题
    subtitle VARCHAR(500),                 -- 副标题
    cover_url VARCHAR(500),                -- 封面图
    course_type VARCHAR(20) NOT NULL,      -- course/video/live/mixed
    category_id UUID,                      -- 分类ID
    teacher_id UUID,                       -- 讲师ID
    description TEXT,                      -- 课程介绍
    target_type VARCHAR(20),               -- all/department/position
    target_values JSONB,                   -- 目标范围
    completion_type VARCHAR(20),           -- duration/complete/exam
    completion_value DECIMAL,              -- 完成条件值
    credit DECIMAL(3,1),                   -- 学分
    duration_minutes INT,                  -- 预计时长
    is_mandatory BOOLEAN DEFAULT false,    -- 是否必修
    status VARCHAR(20) DEFAULT 'draft',   -- draft/published/offline
    created_at TIMESTAMP DEFAULT NOW(),
    updated_at TIMESTAMP DEFAULT NOW()
);
```

#### 4.1.2 章节表
```sql
-- 章节表
CREATE TABLE training_chapters (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    course_id UUID NOT NULL REFERENCES training_courses(id),
    title VARCHAR(200) NOT NULL,
    chapter_type VARCHAR(20) NOT NULL,     -- text/video/live/exam
    sort_order INT DEFAULT 0,
    content TEXT,                          -- 文字内容/富文本
    video_url VARCHAR(500),                -- 视频URL
    video_duration INT,                   -- 视频时长（秒）
    live_start_time TIMESTAMP,             -- 直播开始时间
    live_end_time TIMESTAMP,              -- 直播结束时间
    live_stream_key VARCHAR(100),         -- 推流密钥
    live_record_url VARCHAR(500),          -- 录制回放URL
    required BOOLEAN DEFAULT true,        -- 是否必学
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 3.3.3 学员学习记录表（扩展现有表）
```sql
-- 扩展现有 learning_progress 表
ALTER TABLE training_learning_progress ADD COLUMN:
    chapter_id UUID,                      -- 当前章节
    read_position INT,                    -- 文字阅读位置（字数）
    note_content TEXT,                    -- 学习笔记
    note_time TIMESTAMP,                  -- 笔记时间
    question TEXT,                        -- 提问内容
    question_time TIMESTAMP,              -- 提问时间
    is_commented BOOLEAN DEFAULT false;   -- 是否已评论
```

#### 4.1.4 直播记录表
```sql
-- 直播记录表
CREATE TABLE training_live_sessions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    chapter_id UUID REFERENCES training_chapters(id),
    stream_key VARCHAR(100),               -- 推流密钥
    stream_url VARCHAR(500),              -- 推流地址
    pull_url VARCHAR(500),                -- 拉流地址
    status VARCHAR(20) DEFAULT 'pending', -- pending/live/ended
    viewer_count INT DEFAULT 0,           -- 观看人数
    max_viewer_count INT DEFAULT 0,      -- 最高在线
    total_duration INT,                   -- 总时长（秒）
    record_url VARCHAR(500),              -- 录制文件
    started_at TIMESTAMP,
    ended_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 4.1.5 直播互动表
```sql
-- 直播互动表（弹幕/聊天/问答）
CREATE TABLE training_live_messages (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    session_id UUID REFERENCES training_live_sessions(id),
    user_id UUID NOT NULL,
    message_type VARCHAR(20),             -- danmu/chat/question/answer
    content TEXT,
    is_hidden BOOLEAN DEFAULT false,      -- 是否被隐藏
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 4.1.6 直播签到表
```sql
-- 直播签到表
CREATE TABLE training_live_attendances (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    session_id UUID REFERENCES training_live_sessions(id),
    user_id UUID NOT NULL,
    checkin_time TIMESTAMP,
    duration_watched INT,                 -- 观看时长
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 4.1.7 课程分类表
```sql
-- 课程分类表
CREATE TABLE training_categories (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    name VARCHAR(100) NOT NULL,
    parent_id UUID REFERENCES training_categories(id),
    sort_order INT DEFAULT 0,
    icon VARCHAR(50),
    created_at TIMESTAMP DEFAULT NOW()
);
```

#### 4.1.8 课程评价表
```sql
-- 课程评价表
CREATE TABLE training_reviews (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    course_id UUID REFERENCES training_courses(id),
    user_id UUID NOT NULL,
    rating INT CHECK (rating >= 1 AND rating <= 5),
    content TEXT,
    created_at TIMESTAMP DEFAULT NOW()
);
```

## 七、技术实现方案

### 7.1 视频处理

#### 视频上传与转码
```
前端上传 → 分片上传 → 存储服务 → 触发转码 → 
HLS切片 → CDN分发 → 多清晰度 → 播放器
```

#### 推荐方案
| 方案 | 说明 | 适用场景 |
|------|------|---------|
| 云点播服务 | 阿里云/腾讯云点播 | 正式生产环境 |
| 自建FFmpeg | 需要服务器资源 | 内部培训 |
| 对象存储 + CDN | 存储原始文件 | 成本敏感 |

### 7.2 直播技术

#### 直播架构
```
推流端（OBS/摄像头） → RTMP → 流媒体服务器 → 
    │
    ├── 实时分发（CDN） → 拉流端（播放器）
    │
    └── 录制服务 → 存储 → 回放服务
```

#### 推荐方案
| 方案 | 说明 | 成本 |
|------|------|------|
| 云直播服务 | 阿里云/腾讯云直播 | 按流量计费 |
| 自建SRS | 开源流媒体服务 | 服务器成本 |
| Zoom/腾讯会议 | 第三方会议直播 | 订阅费用 |

### 7.3 文字课程

#### 富文本编辑器
- 使用 **Tiptap** 或 **Slate** 构建
- 支持 Markdown 语法
- 支持图片、表格、代码块
- 支持视频嵌入
