## 完成功能：批量排班与年假规则联动（优先级#4）

### 新增文件
- `server/schedule-engine.ts` - 批量排班与年假规则引擎

### API 路由
- POST `/api/schedule/batch-generate` - 批量生成排班
- POST `/api/schedule/batch-swap` - 批量调班
- POST `/api/schedule/copy` - 复制排班
- GET `/api/shift-types` - 获取班次列表
- POST `/api/shift-types` - 创建班次
- GET `/api/leave/annual-rules` - 获取年假规则
- POST `/api/leave/batch-accrual` - 批量更新年假余额
- GET `/api/leave/employee-info/:id` - 获取员工年假信息
- GET `/api/leave/balances` - 获取年假余额列表

### 功能特性
1. 批量排班：支持选择员工、日期范围、班次类型
2. 排除休息日：支持排除周日/周六
3. 重复模式：每天/每周/每两周循环
4. 年假规则联动：根据入职年限自动计算年假天数
5. 批量更新年假：一键重新计算全员年假余额

### 路由配置
- 排班管理页面 → 批量排班按钮、年假管理按钮

## 完成功能：薪资公式引擎（优先级#3）

- `client/src/pages/SalaryPage/SalaryFormulaPage.tsx` - 公式配置页面

### API 路由
- POST `/api/salary/calculate` - 计算单个员工薪资
- POST `/api/salary/batch-calculate` - 批量计算薪资
- POST `/api/salary/validate-formula` - 验证公式语法
- POST `/api/salary/preview-formula` - 预览公式计算结果

- POST `/api/salary/calculate-insurance` - 计算社保
- POST `/api/salary/items` - 保存薪资项配置
- PUT `/api/salary/items/:id` - 更新薪资项

2. 变量引用：基本工资、岗位工资、绩效工资、考勤数据
3. 个税计算（超额累进税率，专项附加扣除）
4. 社保计算（个人+公司部分）
5. 批量薪资计算
6. 公式预览测试
