## 启动方式

```bash
# 1. 进入项目目录

# 2. 启动后端（端口3000）
npx tsx server/standalone.ts

# 3. 启动前端（端口8081）
npx vite --config vite.config.ts

# 4. 访问系统
http://localhost:8081
```
